# **Reliable Latent-Tree Clustering under Static and Drifting Distributions**
Version 1.0 (Work in progress)

This repository provides the official implementation of the paper "Reliable Latent-Tree Clustering under Static and Drifting Distributions".

## Overview

This project implements algorithms for dynamic and static structural/parameter estimation, along with clustering, for three data types:

1. Tree-structured data generated by latent tree models;
2. Synthetic data generated from Gaussian Mixture Models (GMMs);
3. A real-world dataset.

### Main Scripts:
1. tree_structure_data.py 
2. GMM_data.py
3. driftingPE_tree_structure_data.py
4. real_world_data.py
5. cluster_methods_real.py

#### 1.tree_structure_data.py
This script implements latent variable-based clustering analysis for tree-structured data (Tables 1-2 in the paper).
#### 2.GMM_data.py
This script implements a complete workflow for data generation using a GMM (Table 3 in the paper).

#### 3.driftingPE_tree_structure_data.py

This script implements latent variable-based clustering analysis under drifting data distributions (Figures 3, 4, and 5 of the paper).

#### 4.real_world_data.py and cluster_methods_real.py

These scripts implement real data clustering analysis based on multiple clustering algorithms and evaluates the clustering results using both internal and external metrics (Table 4 in the paper).

## Notes
Methods that integrate Graph Convolutional Networks (GCN) are not reproducible using the code in this repository.
